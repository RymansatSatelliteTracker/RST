# 画面設計書 G-3-1 衛星設定ダイアログ（表示衛星タブ）

## 1. 画面概要

- 画面ID: G-3-1
- 画面名: 衛星設定ダイアログ - 表示衛星タブ
- 役割:
  - 衛星グループへの衛星の登録・並び替え・削除、グループ管理を行う。
- 主な構成:
  - 左エリア（衛星候補リスト・フィルタ検索）
  - 移動ボタン（左→右）
  - 右エリア（グループ選択・グループ内衛星リスト・操作ボタン群）

---

## 2. 画面項目

| ---------- | -------------------------------------------- |

| ②種別      | テキストボックス                             |
| ③表示/入力 | 入力                                         |

### 3.1.2 衛星候補リスト

| 項目       | 内容                                                             |
| ---------- | ---------------------------------------------------------------- |
| ①説明      | 軌道要素データ(OMM)で取得済みの衛星一覧を表示する。形式は「<NORAD ID> 衛星名」。 |
| ②種別      | 仮想スクロールリスト（複数選択可）                               |
| ③表示/入力 | 入力                                                             |

⑤制御条件

## 3. サブダイアログ

### 3.1 衛星情報編集ダイアログ（G-3-1-D1）

衛星候補リストまたはグループ内衛星リストでダブルクリックしたアイテムの周波数・サテライトモード等を設定するダイアログ。

| 項目               | 内容                                                                                    |
| ------------------ | --------------------------------------------------------------------------------------- |
| マニュアル設定表示 | 手動設定済みかどうかを色で示す（緑: 設定済み / グレー: デフォルト）。                   |
| リセットボタン     | デフォルト衛星情報に戻す。                                                              |
| 国際呼称           | 参照用表示（変更不可）。                                                                |
| NORAD ID           | 参照用表示（変更不可）。                                                                |
| 衛星名             | 編集可能なテキストボックス（最大24文字、英数記号のみ）。                                |
| アップリンク周波数 | 周波数（Hz）・運用モードのペアを最大3組設定。自動追尾で使う周波数をラジオボタンで選択。 |
| トーン周波数       | トーン周波数をセレクトボックスで設定。                                                  |
| ダウンリンク周波数 | 周波数（Hz）・運用モードのペアを最大3組設定。自動追尾で使う周波数をラジオボタンで選択。 |
| ビーコン周波数     | 周波数（Hz）・運用モードのペアを1組設定。                                               |
| サテライトモード   | 有効/無効チェックボックスと Normal/Reverse ラジオボタン。                               |
| 概要               | 自由記述テキストエリア（最大1024文字）。                                                |

### 3.2 グループ管理ダイアログ（G-3-1-D2）

グループの追加・名称変更・並び替え・削除を行うダイアログ。

| 項目           | 内容                                                               |
| -------------- | ------------------------------------------------------------------ |
| グループ名入力 | 追加するグループ名を入力するテキストボックス。                     |
| 追加ボタン     | 入力したグループ名でグループを追加する（入力がある場合のみ有効）。 |
| グループリスト | 登録済みグループ一覧（単一選択）。                                 |
| 上移動ボタン   | 選択グループを1つ上に移動。リスト先頭では無効。                    |
| 下移動ボタン   | 選択グループを1つ下に移動。リスト末尾では無効。                    |
| 編集ボタン     | 選択グループ名をインラインダイアログで変更する。                   |
| 削除ボタン     | 選択グループを削除する（グループが1件のみの場合は無効）。          |

### 3.3 ユーザ登録衛星ダイアログ（G-3-1-D3）

ユーザが独自に衛星を登録・編集するダイアログ（TLE文字列または軌道6要素で入力）。

| 項目      | 内容                                                                                                             |
| --------- | ---------------------------------------------------------------------------------------------------------------- |
| 衛星名    | 登録衛星名（最大24文字、英数記号のみ）。                                                                         |
| TLE       | 2行形式のTLE文字列を入力するテキストエリア（最大1024文字）。                                                     |
| OR        | TLEと軌道6要素の区切り表示（どちらか一方を入力する）。                                                           |
| 軌道6要素 | Epoch（UTC）・長半径（km）・RAAN（deg）・離心率・近地点引数（deg）・軌道傾斜角（deg）・平均近点角（deg）を入力。 |

## 4. イベント

### 4.1 表示衛星タブ操作イベント

| No  | 対象項目             | イベント       | 動作                                                                                       |
| --- | -------------------- | -------------- | ------------------------------------------------------------------------------------------ |
| 1   | フィルタテキスト     | 入力           | 衛星候補リストをリアルタイムで絞り込む。                                                   |
| 2   | 衛星候補リスト       | ダブルクリック | 衛星情報編集ダイアログ（G-3-1-D1）を表示する。                                             |
| 3   | 右矢印ボタン         | クリック       | 選択した衛星を右エリアのグループに追加する。重複・上限超えの場合はエラーメッセージを表示。 |
| 4   | グループ選択         | モデル値更新   | グループ内衛星リストを選択グループの衛星リストに切り替える。                               |
| 5   | Groupボタン          | クリック       | グループ管理ダイアログ（G-3-1-D2）を表示する。                                             |
| 6   | グループ内衛星リスト | ダブルクリック | 衛星情報編集ダイアログ（G-3-1-D1）を表示する。                                             |
| 7   | 上移動ボタン         | クリック       | 選択アイテムをリスト内で1つ上に移動する。                                                  |
| 8   | 下移動ボタン         | クリック       | 選択アイテムをリスト内で1つ下に移動する。                                                  |
| 9   | 衛星登録ボタン       | クリック       | ユーザ登録衛星ダイアログ（G-3-1-D3）を表示する。                                           |
| 10  | 削除ボタン           | クリック       | 選択アイテムをグループから削除する。                                                       |

| 7   | 近地点引数 | OK押下時           | 0〜360 deg、小数点以下4桁以内                        | エラーメッセージを表示。 |
| 8   | 軌道傾斜角 | OK押下時           | 0〜180 deg、小数点以下4桁以内                        | エラーメッセージを表示。 |

## 7. 備考

- 衛星グループは1グループあたり最大10件まで衛星を登録できる。
- ユーザ登録衛星（userRegistered = true）は衛星登録ボタンで編集可能。通常衛星は衛星情報編集ダイアログでのみ編集可能。
