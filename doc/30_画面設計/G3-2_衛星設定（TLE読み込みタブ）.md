# 画面設計書 G-3-2 衛星設定ダイアログ（TLE読み込みタブ）

## 1. 画面概要

- 画面ID: G-3-2
- 画面名: 衛星設定ダイアログ - TLE読み込みタブ
- 役割:
  - TLEデータを取得するURLの一覧管理（追加・編集・有効/無効切替・削除）を行う。
- 主な構成:
  - TLE URL リスト（チェックボックス付き編集可能リスト）
  - 追加ボタン / 削除ボタン

---

## 2. 画面項目

### 3.1 TLE URL リスト

| 項目       | 内容                                                     |
| ---------- | -------------------------------------------------------- |
| ①説明      | TLEを取得するURL一覧を表示・編集する。                   |
| ②種別      | リスト（編集可能チェックボックス付きテキストフィールド） |
| ③表示/入力 | 入出力                                                   |
| ④表示条件  | 常時                                                     |

⑤制御条件

- 各行はチェックボックス（有効/無効）とURLテキストフィールドで構成される。

### 3.1.1 有効/無効チェックボックス

| 項目       | 内容                                    |
| ---------- | --------------------------------------- |
| ①説明      | そのURLからのTLE取得を有効/無効にする。 |
| ②種別      | チェックボックス                        |
| ③表示/入力 | 入力                                    |
| ④表示条件  | 常時                                    |

⑤制御条件

- なし

### 3.1.2 URL テキストフィールド

| 項目       | 内容                         |
| ---------- | ---------------------------- |
| ①説明      | TLE取得URLを入力・編集する。 |
| ②種別      | テキストフィールド           |
| ③表示/入力 | 入力                         |
| ④表示条件  | 常時                         |

⑤制御条件

- デフォルト値は "https://"。

### 3.2 追加ボタン（+アイコン）

| 項目       | 内容                                 |
| ---------- | ------------------------------------ |
| ①説明      | TLE URL エントリをリストに追加する。 |
| ②種別      | アイコンボタン                       |
| ③表示/入力 | 入力                                 |
| ④表示条件  | 常時                                 |

⑤制御条件

- なし

### 3.3 削除ボタン（ゴミ箱アイコン）

| 項目       | 内容                                 |
| ---------- | ------------------------------------ |
| ①説明      | 選択中のTLE URL エントリを削除する。 |
| ②種別      | アイコンボタン                       |
| ③表示/入力 | 入力                                 |
| ④表示条件  | 常時                                 |

⑤制御条件

- リスト行が選択されていない場合はボタン無効化。

---

## 3. イベント

### 3.1 タブ初期化イベント

| No  | 対象項目       | イベント | 動作                                                                            |
| --- | -------------- | -------- | ------------------------------------------------------------------------------- |
| 1   | TLE URL リスト | 初期表示 | URLが1件もない場合、空エントリ（URL="https://"、有効=false）を1件自動追加する。 |

### 3.2 TLE読み込みタブ操作イベント

| No  | 対象項目         | イベント | 動作                                                             |
| --- | ---------------- | -------- | ---------------------------------------------------------------- |
| 1   | 追加ボタン       | クリック | 空エントリ（URL="https://"、有効=false）をリスト末尾に追加する。 |
| 2   | 削除ボタン       | クリック | 選択中のエントリをリストから削除し、選択状態をクリアする。       |
| 3   | リスト行         | クリック | クリックした行を選択状態にする。                                 |
| 4   | チェックボックス | 変更     | 当該エントリの有効/無効フラグを更新する。                        |
| 5   | URL テキスト     | 入力     | 当該エントリのURLを更新する。                                    |

### 3.3 OKボタン押下時（親ダイアログ経由）

| No  | 対象項目 | イベント | 動作                                                                                               |
| --- | -------- | -------- | -------------------------------------------------------------------------------------------------- |
| 1   | タブ全体 | OK押下   | 入力チェックを実行し、問題なければ設定を保存する。URLが更新されていれば衛星TLEデータを再取得する。 |

---

## 4. 入力チェック（形式チェック）

| No  | 対象項目       | チェックタイミング | 形式チェック内容                                 | エラー時動作             |
| --- | -------------- | ------------------ | ------------------------------------------------ | ------------------------ |
| 1   | URL リスト全体 | OK押下時           | URLが1件以上登録されていること。                 | エラーメッセージを表示。 |
| 2   | URL テキスト   | OK押下時           | 有効なURLスキーマ形式（https:// 等）であること。 | エラーメッセージを表示。 |

---

## 5. 入力チェック（相関チェック）

| No  | チェックタイミング | 対象項目 | 相関チェック内容                                                         | エラー通知メッセージ     |
| --- | ------------------ | -------- | ------------------------------------------------------------------------ | ------------------------ |
| 1   | OK押下時           | URL      | 変更されたURLに対してHTTPアクセスを行い、TLEとして取得できるか確認する。 | エラーメッセージを表示。 |

---

## 6. 備考

- URLアクセスチェックは、初期表示時のURLと比較して変更がある場合のみ実行する。
- URLアクセスチェックに失敗した場合は保存せずエラーメッセージを表示する。
