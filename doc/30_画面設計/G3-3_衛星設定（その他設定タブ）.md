# 画面設計書 G-3-3 衛星設定ダイアログ（その他設定タブ）

## 1. 画面概要

- 画面ID: G-3-3
- 画面名: 衛星設定ダイアログ - その他設定タブ
- 役割:
  - 衛星パス抽出に使用する最小仰角などの共通設定を管理する。
- 主な構成:
  - 衛星パス抽出最小仰角セレクトボックス

---

## 2. 画面項目

### 3.1 衛星パス抽出最小仰角

| 項目       | 内容                                               |
| ---------- | -------------------------------------------------- |
| ①説明      | 衛星パスを抽出する際に使用する最小仰角を設定する。 |
| ②種別      | セレクトボックス                                   |
| ③表示/入力 | 入力                                               |
| ④表示条件  | 常時                                               |

⑤制御条件

- 選択範囲: 0°〜85°（1度刻み）。

---

## 3. イベント

### 3.1 その他設定タブ操作イベント

| No  | 対象項目             | イベント | 動作                                                             |
| --- | -------------------- | -------- | ---------------------------------------------------------------- |
| 1   | 衛星パス抽出最小仰角 | 値変更   | 変更した最小仰角をリアルタイムで衛星パス抽出サービスへ反映する。 |

### 3.2 OKボタン押下時（親ダイアログ経由）

| No  | 対象項目 | イベント | 動作                                             |
| --- | -------- | -------- | ------------------------------------------------ |
| 1   | タブ全体 | OK押下   | 設定値をアプリケーション設定ファイルに保存する。 |

---

## 4. 入力チェック（形式チェック）

| No  | 対象項目             | チェックタイミング | 形式チェック内容                             | エラー時動作       |
| --- | -------------------- | ------------------ | -------------------------------------------- | ------------------ |
| 1   | 衛星パス抽出最小仰角 | 選択時             | 選択肢リスト内の値であること（選択UIで担保） | 不正値は選択不可。 |

---

## 5. 入力チェック（相関チェック）

| No  | チェックタイミング | 対象項目 | 相関チェック内容 | エラー通知メッセージ |
| --- | ------------------ | -------- | ---------------- | -------------------- |
| -   | -                  | -        | なし             | -                    |

---

## 6. 備考

- 衛星パス抽出最小仰角の変更はOKボタン押下前（値変更時）にリアルタイムで反映される。
  OKボタン押下でアプリケーション設定ファイルへ永続化される。
