# 画面設計書 G-3 衛星設定ダイアログ（共通）

## 1. 画面概要

- 画面ID: G-3
- 画面名: 衛星設定ダイアログ
- 役割:
  - 表示衛星・TLE読み込み先・その他衛星関連設定を管理するダイアログ。
- 主な構成:
  - タブ（表示衛星 / TLE読み込み / その他設定）
  - OKボタン / キャンセルボタン

---

## 2. 画面項目

### 2.1 タブ（表示衛星）

| 項目       | 内容                     |
| ---------- | ------------------------ |
| ①説明      | 表示衛星タブを選択する。 |
| ②種別      | タブ                     |
| ③表示/入力 | 入力                     |
| ④表示条件  | 常時                     |

⑤制御条件

- なし

| 項目       | 内容                        |
| ---------- | --------------------------- |

| ②種別      | タブ                        |
| ③表示/入力 | 入力                        |

⑤制御条件

- なし

| 項目       | 内容                       |
| ---------- | -------------------------- |

---

## 3. イベント

### 3.1 ダイアログ初期化イベント

| No  | 対象項目   | イベント | 動作                                                               |
| --- | ---------- | -------- | ------------------------------------------------------------------ |
| 1   | ダイアログ | 初期表示 | アプリケーション設定ファイルを読み込み、各タブの初期値を設定する。 |

### 3.2 共通操作イベント

| No  | 対象項目         | イベント | 動作                                                                                                                                       |
| --- | ---------------- | -------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| 1   | OKボタン         | クリック | 各タブの入力チェックを実行し、問題なければアプリケーション設定を保存してダイアログを閉じる。TLE URLが更新されていれば衛星TLEを再取得する。 |
| 2   | キャンセルボタン | クリック | 変更を破棄してダイアログを閉じる。                                                                                                         |

---

## 4. 備考

- 各タブの詳細は G-3-1〜G-3-3 を参照。
- ダイアログはpersistentモード（背景クリックで閉じない）。
