# 画面設計書 G-4-1 無線機設定ダイアログ（接続設定タブ）

## 1. 画面概要

- 画面ID: G-4-1
- 画面名: 無線機設定ダイアログ - 接続設定タブ
- 役割:
  - 無線機の機種・シリアル接続情報を設定し、接続テストを行う。
- 主な構成:
  - 左エリア（メーカー / 機種 / シリアルポート / CI-Vアドレス / ボーレート / 接続テスト）
  - 右エリア（IPアドレス / IPポート）

---

## 3. イベント

### 3.1 接続設定タブ操作イベント

| No  | 対象項目           | イベント | 動作                                                           |
| --- | ------------------ | -------- | -------------------------------------------------------------- |
| 1   | メーカー           | 値変更   | 機種リストを更新する。ICOM以外の場合はCI-Vアドレスを空にする。 |
| 2   | 機種               | 値変更   | ICOMの場合、選択機種のデフォルトCI-Vアドレスを設定する。       |
| 3   | シリアルポート更新 | クリック | シリアルポート一覧を最新情報に更新する。                       |
| 4   | 接続テストボタン   | クリック | 選択設定で接続テストを実行し、成功/失敗を通知する。            |
