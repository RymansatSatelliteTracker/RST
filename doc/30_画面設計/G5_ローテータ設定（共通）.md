# 画面設計書 G-5 ローテータ設定ダイアログ（共通）

## 1. 画面概要

- 画面ID: G-5
- 画面名: ローテータ設定ダイアログ
- 役割:
  - ローテータの接続設定・動作設定を管理するダイアログ。
- 主な構成:
  - タブ（接続設定 / 動作設定）
  - OKボタン / キャンセルボタン

---

## 2. 画面項目（共通部）

### 2.1 タブ（接続設定）

| 項目       | 内容                           |
| ---------- | ------------------------------ |
| ①説明      | 接続設定タブを選択する。       |
| ②種別      | タブ                           |
| ③表示/入力 | 入力                           |
| ④表示条件  | 常時                           |
⑤制御条件
  - なし

### 2.2 タブ（動作設定）

| 項目       | 内容                           |
| ---------- | ------------------------------ |
| ①説明      | 動作設定タブを選択する。       |
| ②種別      | タブ                           |
| ③表示/入力 | 入力                           |
| ④表示条件  | 常時                           |
⑤制御条件
  - なし

### 2.3 OKボタン

| 項目       | 内容                                     |
| ---------- | ---------------------------------------- |
| ①説明      | 入力内容を保存してダイアログを閉じる。   |
| ②種別      | ボタン                                   |
| ③表示/入力 | 入力                                     |
| ④表示条件  | 常時                                     |
⑤制御条件
  - なし

### 2.4 キャンセルボタン

| 項目       | 内容                                     |
| ---------- | ---------------------------------------- |
| ①説明      | 変更を破棄してダイアログを閉じる。       |
| ②種別      | ボタン                                   |
| ③表示/入力 | 入力                                     |
| ④表示条件  | 常時                                     |
⑤制御条件
  - なし

---

## 3. イベント

### 3.1 ダイアログ初期化イベント

| No  | 対象項目   | イベント | 動作                                                                                                       |
| --- | ---------- | -------- | ---------------------------------------------------------------------------------------------------------- |
| 1   | ダイアログ | 初期表示 | ローテータAutoをOFFにする。シリアルを切断する。接続設定タブを初期表示し、アプリケーション設定を読み込む。  |

### 3.2 共通操作イベント

| No  | 対象項目         | イベント | 動作                                                                                                       |
| --- | ---------------- | -------- | ---------------------------------------------------------------------------------------------------------- |
| 1   | タブ切替         | クリック | 現在タブの入力チェックを実行し、エラーがあればタブを切り替えない。                                         |
| 2   | OKボタン         | クリック | 現在タブの入力チェックを実行し、問題なければ設定を保存してダイアログを閉じる。シリアル接続を再開する。     |
| 3   | キャンセルボタン | クリック | 変更を破棄してダイアログを閉じる。シリアル接続を再開する。                                                 |

---

## 4. 備考

- 各タブの詳細は G-5-1〜G-5-2 を参照。
- ダイアログ表示時はローテータAuto制御をOFFにし、シリアル接続を切断する。
- OK・キャンセルいずれの場合も、ダイアログを閉じる際にシリアル接続・ローテータ状態監視を再開する。
- ESCキーでもダイアログを閉じることができる（キャンセルと同等）。
